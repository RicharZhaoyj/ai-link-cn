# AiCoin走势分析操作指南

## 1. 打开AiCoin软件
- 双击桌面上的AiCoin图标
- 或从开始菜单找到AiCoin

## 2. 查看行情页面
1. **点击"行情"或"Market"选项卡**
2. **搜索感兴趣的加密货币** (BTC, ETH, SOL等)
3. **查看实时价格**和图表

## 3. 使用分析工具
AiCoin通常有以下分析功能：

### **技术指标**
- **移动平均线 (MA)**
  - 设置7日、30日移动平均线
  - 观察交叉点判断趋势
  
- **RSI相对强度指数**
  - 当RSI < 30时可能超卖（买入机会）
  - 当RSI > 70时可能超买（卖出机会）

- **MACD指标**
  - 查看MACD线和信号线交叉
  - 柱状图颜色变化判断买卖信号

- **布林带 (Bollinger Bands)**
  - 观察价格是否突破上轨或下轨
  - 布林带收缩表示波动减少

### **图表工具**
1. **选择时间周期**
   - 1小时、4小时、日线、周线
2. **添加指标**
   - 点击图表上的"添加指标"按钮
3. **查看成交量**
   - 成交量柱状图配合价格分析

## 4. 具体分析步骤

### **比特币(BTC)分析**
```
1. 搜索"BTC/USDT"
2. 选择日线图表
3. 添加以下指标:
   - MA7 (7日移动平均线)
   - MA30 (30日移动平均线)
   - RSI (14日周期)
   - MACD
   - 布林带
```

### **以太坊(ETH)分析**
```
1. 搜索"ETH/USDT"
2. 选择4小时图表
3. 添加以下指标:
   - MA15 (15小时移动平均线)
   - MA60 (60小时移动平均线)
   - RSI (14周期)
   - MACD
```

## 5. 分析要点

### **趋势判断**
- **上涨趋势**：价格 > MA30，MACD柱状图为正
- **下跌趋势**：价格 < MA30，MACD柱状图为负
- **震荡趋势**：价格在布林带中部徘徊

### **买卖信号**
- **买入信号**：
  - RSI < 30 (超卖)
  - MACD从负转正
  - 价格突破布林带上轨
  
- **卖出信号**：
  - RSI > 70 (超买)
  - MACD从正转负
  - 价格跌破布林带下轨

## 6. 风险管理

### **止损设置**
```
建议止损位置：
- 价格跌破MA30时考虑止损
- RSI超过75时考虑减仓
- 成交量异常放大时注意风险
```

### **仓位管理**
```
建议仓位配置：
- 比特币(BTC)：40-50%仓位
- 以太坊(ETH)：20-30%仓位
- 其他加密货币：10-20%仓位
```

## 7. 实时监控

### **警报设置**
在AiCoin中设置价格警报：
```
1. 点击"警报"或"Alert"功能
2. 设置BTC价格超过$70,000警报
3. 设置ETH价格低于$3,000警报
```

### **关注点**
```
需要关注的指标：
1. 成交量变化 - 是否异常
2. 价格突破关键支撑/阻力位
3. RSI持续处于极端位置
4. MACD出现背离信号
```

## 8. 导出分析

### **截图保存**
```
1. 分析完成后截图保存
2. 截图保存路径：~/AiCoin_analysis/日期.png
```

### **报告导出**
```
1. 点击"导出"或"Export"功能
2. 选择"PDF"或"Excel"格式
3. 保存分析报告
```
