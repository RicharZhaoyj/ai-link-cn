Midjourney完整教程

从零开始掌握AI图像生成，成为AI绘画大师

⏱️ 阅读时间：45分钟 📊 难度：初级到高级 📝 更新：2026.07.22 👁️ 3.8万次学习

一、Midjourney基础入门

1.1 什么是Midjourney

Midjourney是一款基于AI的图像生成工具，通过文字描述（Prompt）就能生成高质量的艺术图像。它以卓越的画质和丰富的风格控制能力著称，是设计师、艺术家和创意工作者的首选AI绘画工具。

1.2 快速开始

  1. 注册Discord账号：Midjourney通过Discord使用，先注册Discord账号
  2. 加入Midjourney服务器：访问 midjourney.com，点击Join the Beta
  3. 选择订阅计划：基础版$10/月起，标准版$30/月，专业版$60/月
  4. 开始生成图像：在Newbies频道输入 /imagine + 描述文字
💡 新手建议

刚开始可以先用基础版熟悉操作。当你需要更多生成次数和更高画质时，再考虑升级。新手建议从Standard版开始，性价比最高。

1.3 生成你的第一张图

输入以下Prompt，体验Midjourney的魅力：

示例 Prompt
/imagine prompt: a beautiful sunset over the ocean, golden hour, cinematic lighting, highly detailed, 8k --ar 16:9 --v 6.0

这个Prompt包含：

  • 主体描述：a beautiful sunset over the ocean
  • 风格/氛围：golden hour, cinematic lighting
  • 质量描述：highly detailed, 8k
  • 参数：--ar 16:9（宽高比）, --v 6.0（版本）

二、提示词结构详解

2.1 基础结构公式

一个好的Midjourney Prompt通常包含以下几个部分：

Prompt 结构公式
[主体] + [场景/环境] + [风格] + [光影] + [视角] + [画质描述] + [参数]

详细说明：

  • 主体（Subject）：你想画什么 - 人物、动物、风景、物品等
  • 场景/环境（Environment）：在什么地方 - 森林、城市、太空、室内等
  • 风格（Style）：艺术风格 - 油画、水彩、赛博朋克、宫崎骏风格等
  • 光影（Lighting）：光线效果 - 黄金时刻、霓虹灯光、柔和光线等
  • 视角（View）：拍摄角度 - 特写、鸟瞰、平视、广角等
  • 画质（Quality）：质量描述 - 高清、精细、8k、照片级等

三、常用参数详解

3.1 宽高比 --ar

控制生成图像的宽高比例：

参数比例适用场景
--ar 1:1正方形头像、图标、社交媒体配图
--ar 16:9横版16:9壁纸、视频封面、横幅
--ar 9:16竖版9:16手机壁纸、短视频封面
--ar 4:34:3传统照片比例
--ar 3:23:2单反照片比例
--ar 2:32:3竖版照片比例

3.2 质量与细节参数

  • --q 0.25：粗糙质量，生成速度快，适合快速试验想法
  • --q 0.5：中等质量，平衡速度和质量
  • --q 1：默认质量，日常使用推荐
  • --q 2：高质量，细节更丰富，消耗2倍时长
  • --hd：高清模式，适合建筑和风景
  • --style raw：原始风格，减少艺术美化

3.3 风格化程度 --s

控制Midjourney对图像的风格化程度：

  • --s 0：完全按照描述，不添加额外风格
  • --s 100：低风格化，比较写实
  • --s 250：默认值，平衡风格和描述
  • --s 500：高风格化，艺术性更强
  • --s 750：很高风格化，创意性强
  • --s 1000：最高风格化，非常艺术化
🎯 使用技巧

想要图像更接近你的描述，用较低的--s值；想要更有艺术感和创意，用较高的--s值。人像摄影建议s值在100-250之间，艺术创作可以尝试500-1000。

3.4 其他实用参数

  • --no：排除某些元素，如 --no text, watermark
  • --seed：指定种子值，生成相似图像
  • --chaos：增加随机性，0-100，值越高变化越大
  • --tile：生成可平铺的图案纹理
  • --iw：图像权重，图生图时控制参考图影响程度

四、高级技巧

4.1 图生图技巧

上传图片作为参考，可以更好地控制生成效果：

  1. 将图片拖入Discord发送
  2. 右键图片，复制图片链接
  3. 在Prompt开头粘贴图片URL
  4. 后面跟上文字描述和参数
  5. 使用 --iw 0.5-2 调整参考图权重
图生图 Prompt
[图片URL] cyberpunk city, neon lights, rainy night, cinematic --iw 1.5 --ar 16:9 --v 6.0

4.2 Remix模式

开启Remix模式后，可以在变体（Vary）时修改Prompt：

  1. 输入 /settings 打开设置面板
  2. 点击 Remix mode 开启
  3. 生成图像后点击 V1/V2/V3/V4
  4. 弹出的对话框中修改Prompt
  5. 点击提交生成新变体

4.3 局部重绘（Vary Region）

只修改图像的某一部分，保持其他部分不变：

  1. 放大（Upscale）一张图
  2. 点击 Vary (Region) 按钮
  3. 用画笔涂抹要修改的区域
  4. 输入新的描述文字
  5. 点击提交生成
✨ 实用场景

局部重绘非常有用：人物换发型、添加/移除物体、修改服装、换背景中的元素等。配合Remix模式效果更好。

4.4 缩放扩展（Zoom Out）

扩展图像边界，生成更大的场景：

  • Zoom Out 2x：2倍缩放，扩展画面
  • Zoom Out 1.5x：1.5倍缩放，小幅扩展
  • Custom Zoom：自定义缩放比例和新Prompt

配合Pan（上下左右平移）功能，可以制作超大幅全景图。

六、最佳实践与避坑

6.1 写出好Prompt的7个秘诀

  1. 具体明确：用具体名词替代模糊描述，"布偶猫"比"猫"好
  2. 加入细节：材质、纹理、环境细节越多越好
  3. 描述光线：光线决定画面氛围，黄金时刻、柔光、逆光等
  4. 指定视角：特写、广角、俯视、平视等
  5. 添加质感：照片级、油画、数字艺术、像素风等
  6. 参考艺术家：加入艺术家名字快速定位风格
  7. 多迭代优化：不断调整Prompt，逐步逼近理想效果

6.2 常见问题与解决

  • 图像模糊不清晰：加入highly detailed, sharp focus, 8k等词
  • 人物变形：增加--q 2，使用photorealistic，指定相机和镜头
  • 文字乱码：使用--no text，或后期用其他工具添加文字
  • 手指异常：加入perfect hands, detailed fingers，或后期修复
  • 风格不对：加入具体艺术家名或风格关键词，调整--s值
  • 太抽象看不懂：减少--s值，增加具体描述，使用--style raw

6.3 高效工作流

专业的AI绘画工作流程：

  1. 灵感收集：收集参考图，确定风格和主题
  2. 快速草图：用--q 0.25快速生成大量方案
  3. 筛选优化：选满意的方向，优化Prompt
  4. 精细调整：使用Remix和Vary Region微调细节
  5. 高清输出：Upscale放大，必要时用Zoom Out扩展
  6. 后期处理：Photoshop微调、修瑕疵、调色
