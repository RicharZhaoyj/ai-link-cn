## 📅 每周内容发现流程

### 周一：工具发现日
```
任务:
1. 浏览主要AI工具聚合平台 (30分钟)
2. 记录发现的新工具 (10-15个)
3. 快速筛选有潜力的工具 (3-5个)
4. 初步调研这些工具的功能和特点
产出: 本周潜在评测工具列表
```

### 周二：深度调研日
```
任务:
1. 对筛选出的工具进行深度调研
2. 收集以下信息:
   - 工具功能特点
   - 定价方案
   - 用户评价
   - 市场份额和竞品
   - 技术特点和创新点
3. 确定评测角度和价值点
产出: 详细的工具调研报告
```

### 周三：内容规划日
```
任务:
1. 确定评测文章的框架结构
2. 制定评测维度和测试计划
3. 准备需要的测试数据或案例
4. 规划内容发布和推广策略
产出: 评测内容大纲和计划
```

### 周四：内容创作日
```
任务:
1. 开始撰写评测文章
2. 创建必要的图表和对比表格
3. 拍摄或制作演示截图
4. 确保内容深度和实用性
产出: 完成80%的评测内容
```

### 周五：完善和发布日
```
任务:
1. 完成剩余20%内容
2. 优化SEO和用户体验
3. 添加内部链接和相关推荐
4. 准备社交媒体推广材料
5. 发布到网站并开始推广
产出: 发布的评测页面和推广计划
```

## 📊 工具评估标准

### 核心评估维度 (每个工具都要评估)

| 维度 | 评估内容 | 权重 |
|------|----------|------|
| **功能完整性** | 核心功能、特色功能、易用性 | 30% |
| **性能表现** | 响应速度、准确性、稳定性 | 20% |
| **定价策略** | 价格合理性、免费版功能、性价比 | 15% |
| **用户评价** | 用户满意度、评价一致性、问题反馈 | 15% |
| **市场潜力** | 增长趋势、市场份额、创新性 | 10% |
| **技术深度** | 技术架构、AI模型、算法创新 | 10% |

### 优先级评分系统 (1-10分)
```
8-10分: 立即评测 - 具有重大影响力或创新性
6-7分: 高优先级 - 值得关注的优秀工具
4-5分: 中等优先级 - 在特定领域有价值
1-3分: 低优先级 - 可能不值得评测
```
