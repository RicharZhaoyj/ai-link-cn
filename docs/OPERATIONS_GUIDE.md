# AI.link.cn 完整运营指南

## 🎯 项目概述

**AI.link.cn** 是一个专注于AI工具评测和推荐的平台，通过联盟营销（Affiliate Marketing）实现盈利。

### 核心目标
1. **短期目标**（3个月）：月收入达到 $3,000
2. **中期目标**（6个月）：月收入达到 $5,000
3. **长期目标**（12个月）：月收入达到 $10,000

### 收入模型
- **主要收入**: Affiliate佣金（80%）
- **次要收入**: 广告和赞助（15%）
- **其他收入**: 增值服务（5%）

## 🏗️ 网站架构

### 核心页面
```
/
├── index.html                    # 网站首页
├── pages/                        # 内容页面
│   ├── tools/                    # 工具评测页面
│   │   ├── hostinger.html       # Hostinger详细评测
│   │   ├── chatgpt.html         # ChatGPT评测（待建）

└── config/                       # 配置文件
    ├── affiliate_links.json     # Affiliate链接配置
    └── tools_list.json          # 工具列表配置
```

### 技术栈
- **前端**: 静态HTML + CSS + JavaScript
- **样式框架**: Pico.css (轻量级CSS框架)
- **图表库**: Chart.js (数据可视化)
- **部署**: Vercel (免费托管)
- **监控**: 自定义监控面板

## 🔄 日常工作流程

### 每日任务 (约1小时)
1. **检查收入** (10分钟)
   - 登录联盟监控面板
   - 查看昨日点击和转化数据
   - 记录收入变化

2. **检查链接** (10分钟)
   - 测试所有Affiliate链接
   - 更新失效链接
   - 检查页面加载速度

3. **内容更新** (30分钟)
   - 发布1-2篇简短内容
   - 更新工具价格信息
   - 回复用户评论

4. **SEO监控** (10分钟)
   - 检查关键词排名
   - 分析流量来源
   - 调整SEO策略

### 每周任务 (约3小时)
1. **运行优化脚本** (30分钟)
   ```bash
   cd /root/.openclaw/workspace
   node scripts/affiliate-optimizer.js
   ```

2. **生成周报** (1小时)
   - 分析每周表现
   - 制定下周计划
   - 调整优化策略

3. **内容创作** (1.5小时)
   - 撰写1篇深度评测文章
   - 创建相关教程内容
   - 优化现有页面

### 每月任务 (约6小时)
1. **收入对账** (2小时)
   - 核对各平台佣金
   - 准备税务文件
   - 分析月度趋势

2. **策略调整** (2小时)
   - 评估联盟项目表现
   - 决定是否继续合作
   - 寻找新的联盟机会

3. **技术维护** (2小时)
   - 网站备份
   - 代码优化
   - 性能测试
